Motion System

Every animation has a purpose. Hover to demo. Use prefers-reduced-motion to disable non-essential animations.

Entrance Reveal

translateY(16px) → 0, opacity 0 → 1, 400ms ease-out. Scroll-triggered via IntersectionObserver.

Duration Scale
150ms — hover, focus
250ms — card, button
400ms — reveal, page
600ms — hero, modal
Easing Curves
ease-out

ease-spring

ease-in-out
cubic-bezier(0.4, 0, 0.2, 1)

Reduced Motion — Always Honor
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
